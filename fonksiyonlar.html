Fonksiyonlar

Orta ve büyük ölçekli bir programda fonksiyonlar programın temel yapı taşıdır. C++ fonksiyonları, programın belirli işlemini gerçekletirir ve küme parantezi ‘{ }’ ile başlayıp sonlanır. Yani, programı fonksiyonlar oluşturur.

  • C++ fonksiyonları bir program içerisinde tekrar tekrar aynı mantık içerisinde yeniden yazmamak için kullanılır
  • Gerekli her yerde aynı işlevi sağlayacak kodlar yazmamak için istenildiği kadar fonksiyonlar çağrılır ve kullanılır.
  • Fonksiyonlara ayrılmış büyük bir C++ programı kolayca anlaşılır, geliştirilir ve bakımı kolay olur.
  • Büyük bir işi yapan bir fonksiyon küçük fonksiyonlara ayrılarak temel görevlere ayrılabilir.

/*
 * İki Sayiyi Toplayan Fonksiyon (ToplamaFonksiyonOrnegi.cpp)
 */
#include <iostream> 

using namespace std;

int Topla(int sayi1,int sayi2) { // int türünden bir sonuç döndürmek istediğimiz için int yazdık 
     int Sonuc = sayi1 + sayi2 ;
     return Sonuc;  // Bir değer döndürmemiz için return yazmamız gerekli
}

int main() {
int sayi = Topla(5,3);  // 5 ile 3 ü parametre olarak gönderdik ve toplamasını istedik. Fonksiyon sonucu onu çağırdığımız yere geri döndürecek
cout <<" Sonuc : " <<  sayi  << endl ;
return 0;                       
}

Kodun Çıktısı: 

    Sonuc = 8

Fonksiyonlar, main fonksiyonundan sonra da tanımlanabilir. Ancak bunun için main fonksiyonundan önce bazı temel tanımlamalar yapılması gerekir. Fonksiyonun türü,ismi ve aldığı parametrelerin belirtilmesi gerekir. Bu tanımlamalar yapıldıktan sonra artık main fonksiyonundan sonra detaylı bir şekilde fonksiyon yazılabilir.

/*
 * Girilen Sayının Faktöriyelini Bulan Fonksiyon(FaktoriyelFonksiyon.cpp)
 */
#include <iostream> 

using namespace std;

int Faktoriyel(int);

int main() {

cout<<" Faktoriyelini istediginiz sayi : ";
int  sayi =0;
cin>>sayi;
int Islem = Faktoriyel(sayi);
cout<<" Sonuc : "<< Islem;
return 0;
}

int Faktoriyel(int GirilenSayi) {
int Sonuc = 1;
while(GirilenSayi>1){
Sonuc *= GirilenSayi;
GirilenSayi--;
}
return Sonuc; 
}

Kodun Çıktısı: 

Faktoriyelini istediginiz sayi : 5
Sonuc : 120
