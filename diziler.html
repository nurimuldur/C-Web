Diziler

C++ dilinde diziler, aynı veri tipindeki değişkenlerin toplandığı yapılar olarak adlandırılır.

  • Diziler herhangi bir veri tipinde oluşturulabilir.
  • Dizi boyutu sabit bir değer olmalıdır.
  • Dizi elemanları daima bellekte bitişik adres yerlerine yazılır.
1
2


int a[10];  // integer(tam sayi) tipinde bir dizi
char b[10];  // char(karakter) tipinde bir dizi

/*
 * Dizi Elemanlarını Yazdırma (DiziElemanYazdirma.cpp)
 */
#include <iostream> 

using namespace std;

int main() {
int a[6]={0,1,2,3,4,5};  // dizi tanımlaması

for(int i=0;i<=6;i++){ 

cout <<"Dizinin" << i+1<< ". elemani: " << a[i] << endl;  // dizi elemanları ekrana yazdırılıyor 
}
return 0;                       
}

Kodun Çıktısı: 

Dizinin 1. elemani : 0
Dizinin 2. elemani : 1
Dizinin 3. elemani : 2
Dizinin 4. elemani : 3
Dizinin 5. elemani : 4
Dizinin 6. elemani : 5

/*
 * İki Boyutlu Dizi Elemanlarını Yazdırma (İkiBoyutluDiziElemanYazdirma.cpp)
 */
#include <iostream> 

using namespace std;

int main() {
int dizi[2][2]={{2,4},{3,5}};  // dizi tanımlaması

for(int i=0;i<2;i++){

   for(int j=0;j<2;j++){ 

cout <<"Dizinin" << i << "x" << j << "indisli elemani: " << dizi[i][j] << endl;  // dizi elemanları ekrana yazdırılıyor 
} }
return 0;                       
}

Kodun Çıktısı: 

Dizinin 0x0 indisli elemani: 2
Dizinin 0x1 indisli elemani: 4
Dizinin 1x0 indisli elemani: 3
Dizinin 1x1 indisli elemani: 5
