Döngüler

Algoritma kurarken ve kod yazarken işimiz hızlandıracak ve kolaylaştıracak unsurlardan biride döngülerdir. Döngüler kontrol koşulu doğru olduğu sürece dönmektedir. Kontrol koşulu sağlanmadığı taktirde döngüden çıkmaktadır. C programlama dilinde 3 tip döngü vardır:

  • For
  • While
  • Do-While

For Döngüsü

for(Başlangıç,Koşul,Değişim)

Başlangıç: Genellikle bir ya da daha çok tanımlanmış bir başlangıç değeridir. Döngünün başlayacağı değeri göstermektedir.

Koşul: Burada koşul hesaplanır ve eğer koşul tutuyorsa değişim miktarına geçer. Eğer koşul tutmuyorsa döngüden çıkar.

Değişim: Koşulun doğru olduğu sürece başlangıç değerinin değişim miktarını belirtmektedir.

/*
 * For döngüsü örneği (ForDongusu.cpp)
 */
#include <iostream> 

using namespace std;

int main() {
int x;  // x adında int tipinde bir değişken
int i;  // i adında int tipinde bir değişken(Bu değer bizim döngü sayacımız olacak)
x=0;  // x değişkenine bir değer atmak
for(i=1;i<=5;i++){  // for döngüsü oluşturmak
cout << x << "\n" << endl ;  // x değişkenini ekrana yazdırmak                  
x=x+2;  // Döngü her çalıştığında x değişkenini 2 arttırmak                 
}
return 0;                       
}

While Döngüsü

İkinci döngü türümüz olan while döngüsü kullanış amacı olarak aslında for döngüsünden çok farklı olduğunu söyleyemeyiz. İçerideki koşul sağlandığı sürece döngü dönmektedir. Şöyle gösterecek olursak while(Koşul).

/*
 * While döngüsü örneği (WhileDongusu.cpp)
 */
#include <iostream> 

using namespace std;

int main() {
int n;  // n adında int tipinde bir değişken
int toplam=0;  // toplam adında int tipinde bir değişken ve ilk değeri 0                  
int kalan;  // kalan adında int tipinde bir değişken
           
cout << "Bir Sayi Giriniz: ";   // Giriş istenmesi
cin >> n;                   // Girilen sayıyı n değişkenine atmak

while (n!=0){  // n 0'a eşit değilse                                              
kalan = n % 10; // n değişkeninin mod10'unu alıp kalan değişkenine atmak
toplam = toplam + kalan; // toplam değişkeniyle kalan değişkenini toplayıp toplam değişkenine atmak
n = n / 10; // n değişkenini 10 değerine bölüp n değişkenine atmak        
}
     
cout << "Girilen Sayinin Basamaklar Toplami: " << toplam << endl;  
return 0;
}

Kodun Çıktısı: 

Bir Sayi Giriniz: 123
Girilen Sayinin Basamaklar Toplami: 6

/*
 * Do-While döngüsü örneği (DoWhileDongusu.cpp)
 */
#include <iostream> 

using namespace std;

int main() {
int i=1;  // i adında int tipinde bir değişken ve ilk değeri 1

do{

cout << "i'nin değeri: " << i << endl;
i++;

}while(i<=4 && i>=2);
return 0;
}
