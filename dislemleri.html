Dosyalama İşlemleri

C++ ile dosya işlemleri yapabilmek için "#include <fstream>" satırını eklememiz gerekir. Bu satır stream kütüphanesini programa dahil eder.

C++ ile dosyaya yazı yazmak için "ofstream" sınıfını kullanmak gerekir.

C++ ile yazılı olan bir dosyadan veri okumak için "ifstream" sınıfını kullanmak gerekir.

Bu dosyalama işlemleri gerçekleştirmek için fiziksel olarak bir dosya oluşturmamız lazım.Dosya oluşturmak için"open(dosyaadi,mod);"
Burada mod dosyayı hangi amaç için açtığımız için gereklidir.

  • ios::in (Veri okuma modu)
  • ios::out (Veri yazma modu)
  • ios::app (Veri ekleme modu)

Adımlar

  • Önce seçeceğimiz sınıftan bir nesne yaratıyoruz.
  • Bu nesneden open() fonksiyonu kullanarak bir dosya oluşturuyoruz ve modunu belirliyoruz.
  • Dosya açma kalıbımız şu şekilde; Nesneİsmi.open(Dosyaİsmi, Mod)
  • İşlemlerimizi yapıyoruz.
  • close() fonksiyonunu kullanarak dosyamızı kapatıyoruz.

/*
 * Dosya işlemleri kayıt örneği (DosyaIslemleriOfstream.cpp)
 */
#include <iostream> 
#include <fstream> 

using namespace std;

int main() {
                    
ofstream dosya1;
dosya1.open("isim.txt");
dosya1 << "Isım kayit dosyasi:\n";
dosya1.close(); // dosya kapatma
cout << "Dosyaya kayit yapildi" << endl;
system("Pause");                        
return 0;                       
}

/*
 * Dosya işlemleri okuma örneği (DosyaIslemleriIfstream.cpp)
 */
#include <iostream> 
#include <fstream> 
#include <string> 

using namespace std;
string ad; 

int main() {
                    
ifstream dosya1("isimler.txt",ios::in);
if(dosya.is_open())
{
do
{
dosya1 >> ad;
cout << ad << endl;
} while(!dosya.eof());  // eof(EndOfFile) dosya sonuna kadar demek
dosya1.close();
cout << "Dosyada adlar okundu" << endl;
}
else
cout << "Dosya acilmadi";
system("Pause");                        
return 0;                       
}

/*
 * Dosya işlemleri arama örneği (DosyaIslemleriArama.cpp)
 */
#include <iostream> 
#include <fstream> 
#include <string> 

using namespace std;
string ad; 

int main() {
                    
ifstream dosya1("isimler.txt",ios::in);
string arananisim;
cout << "Aranacak isim: " << endl;
cin >> arananisim;
if (dosya1.is_open()) {
int x=0;
do {
dosya1 >> ad;
if (ad==arananisim) {
cout << "Aranan isim dosyada mevcuttur." << endl; 
x=1;
} while(!dosya1.eof());
dosya1.close();
if(x ==0)
cout << "Dosyada aranan isim mevcut degil" << endl;
}
else
cout << "Dosya acilmadi";
system("Pause");                        
return 0;                       
}
